# Windows Components

## Cygwin Installation

For Windows installations, Cygwin is required for script execution.

### Download and Install

1. Download from [www.cygwin.com](http://www.cygwin.com)
2. Run setup.exe
3. Select defaults:
   - Install from Internet
   - Install Root: C:\cygwin
   - Install for All Users
4. Select packages directory
5. Configure proxy if needed
6. Select mirror site

### Windows Service Registration

Register Tomcat as a Windows service using service.bat script.
